🎯 ¿Para qué sirven?

¿Para qué se usan las variables?

Una variable es como una cajita con un nombre, donde el proyecto guarda algo que puede cambiar mientras se ejecuta. Estos son ejemplos de la vida real de un juego o proyecto:

🏆

Puntaje

Guarda cuántos puntos ganó el jugador. Sube cada vez que atrapa algo o acierta una respuesta.

❤️

Vidas

Guarda cuántas vidas le quedan al personaje. Baja cada vez que choca con un enemigo.

⏱️

Tiempo restante

Guarda los segundos que quedan en un cronómetro. Baja solo, de a poco, mientras el juego corre.

🙋

Nombre del jugador

Guarda el nombre que escribió la persona que está jugando, para saludarla o mostrar su puntaje.

📶

Nivel

Guarda en qué nivel o fase del juego está el jugador, para saber qué mostrar en pantalla.

🎨

Color elegido

Guarda qué color o personaje eligió el jugador al principio, para usarlo durante todo el juego.

En todos los casos, la idea es la misma: algo puede cambiar (subir, bajar, o reemplazarse), y la variable es el lugar donde el proyecto se acuerda de ese valor.

¿Qué es una variable?

🧠 Actividad 1 · 2.5 pts

✅❌ Verdadero o falso

Una variable guarda un solo valor a la vez.

El bloque "sumar a variable 1" reemplaza el valor anterior, no le suma nada.

El bloque "dar a variable el valor" reemplaza el valor anterior.

Una variable puede guardar varios valores a la vez, uno al lado del otro.
