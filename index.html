✅❌ Verdadero o falso

Una variable guarda un solo valor a la vez.

El bloque "cambiar variable en 1" reemplaza el valor anterior, no le suma nada.

El bloque "fijar variable a" reemplaza el valor anterior.

Una variable puede guardar varios valores a la vez, uno al lado del otro.

Una lista puede guardar varios valores en distintas posiciones.

fijar contador a 0
repetir 3 veces [ cambiar contador en 2 ]

¿Cuánto vale contador al final?

El bloque "preguntar y esperar" no guarda la respuesta en ningún lado, se pierde apenas se escribe.

¿Qué bloque hay que usar para que lo que escribe el usuario quede guardado en una lista?

Si no usás "eliminar todos los elementos" antes de repetir el bloque preguntar + agregar, la lista se sigue llenando con datos viejos cada vez que se ejecuta el script.
